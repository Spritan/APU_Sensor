About Us

The open sensor network was developed as a collaborative network that helps increase data coverage across India

Our Mission

Through this project we hope to be able to reach areas where environment monitoring is lacking, or non existent. Having created a viable dataset, this can be used to further aims and aspirations of locals and help to shed light on problems and challenges that people may face from the environment

The objectives of the sensor network are

  • To display data collected from the open sensor network
  • To act as an independent web portal for learning about local meteorological and environmental conditions
  • To act as a database for researchers to access data on environmental parameters
  • To allow citizen scientists to connect to the sensor network, learn more and begin contributing to the platform
  • To allow for API connections to allow direct connections with the database for interested parties

Frequently Asked Questions

The project is aimed at generating a dataset on environmental parameters using off-the-shelf sensors. The sensor node is easily built and deployed and starts sending data to the server.

The data is obtained from our own sensor nodes that are deployed at various places. This data is directly sourced from the nodes themselves and is not tweaked further apart from minor post processing before display on the dashboard.

The sensor node is not calibrated at the moment. Thus there may be errors in the data displayed.

The project accepts new members who can set up their own nodes for environment monitoring.
